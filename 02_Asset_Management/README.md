# 02 Asset Management

## Purpose
This folder is for tracking assets, categories, status, and asset details.

## Simple Implementation Plan
- Create asset records with name, type, and status.
- Add features to add, edit, delete, and view assets.
- Track ownership and availability.

## How to Use
- Place all asset-related code and screens here.
- Keep each asset feature simple and easy to test.
- Connect this module with the database when ready.
